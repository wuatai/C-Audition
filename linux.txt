一、Linux常用命令
1.1 命令基本格式
命令 [选项]	[参数]
权限：-rw-r--r--
文件类型(-文件，d目录，l软连接文件)
2-4,所有着
5-7,所属组
8-10,所有人
r读，w写，x执行
1.2	文件处理命令
目录处理命令
mkdir -p [目录名]，-p 递归创建
cd	[目录]，change dir
pwd ，显示当前位置
rmdir [目录名],只能删除空白目录
rm -rf [文件或目录],
cp [选项] [文件或目录],-r 复制目录,-p 连带文件属性复制, -a相当于-rp
mv [文件]	移除文件

bin:系统命令目录
boot:开机相关目录
cgroup:用户组目录
etc:配置文件目录
home:家目录
lib:函数库目录
media:空目录，外界，用于挂载
misc:空目录，外界，用于挂载
mnt:空目录，外界，用于挂载
proc,sys:不能直接操作，保存的是内存过载点
tmp:临时目录
usr:系统软件资源目录
var:系统可见文档目录
proc:直接写入内存的

链接命令
ln -s [源文件] [目标文件] 软连接，不加-s则是硬链接

1.3	文件搜索命令
locate：在后台数据库搜索，速度快，但是不是实时更新的，每天更新一次，所以最新的文件查找不到（执行updatedb可以强制更新）
whereis与which：只能搜索系统命令
find： find [搜索范围] [搜索条件]
	比如：find / -name install.log
字符串搜索命令grep:grep [选项] 字符串 文件名，-i:忽略大小写，-v:排除指定字符串
find命令与grep命令的区别
1.4	帮助命令
man ls --help
help cd(shell命令)
1.5	压缩和解压缩命令
.zip格式压缩
zip 压缩文件名 源文件
zip -r 压缩文件名 源目录
uzip 压缩文件名
.gz格式压缩
gzip -c 源文件 > 压缩文件名
gzip -r 目录：目录中的子文件被压缩
gzip -d 压缩文件名

打包命令tar
tar -cvf 打包文件名 源文件，如tar -cvf jp.tar jp,gz jp.tar
解包命令
tar -xvf 打包文件名,如tar -xvf jp.tar
直接打包压缩
tar -zcvf 打包文件名
解包解压缩命令
tar -xcvf 打包文件名 -C 解压位置
1.7	其他常用命令
挂载命令：
mount [-t 文件系统] [-o 特殊选项] 设备 文件名 挂载点

二、Shell基础
1.Shell概述
Shell是一个命令显示器
2.脚本执行方式
	echo 输出命令
	echo [选项] [输出内容]，-e 可以支持控制字符
	方式：赋予执行权限，直接运行
		chmod 755 hello.sh		./hello.sh
		bash hello.sh
3.别名与快捷键
	alias:显示所有别名
	alias ls='ls --color=auto'，alias [别名] [原命令]
	unalias:删除别名
	
	ctrl+c 强制终止当前命令
	ctrl+l 清屏
	ctrl+a 光标移动到命令行首
	ctrl+e 光标移动到命令行尾
	ctrl+u 从光标所在位置删除到行首
	ctrl+z 把命令放到后台
	ctrl+r 在历史命令中搜索
4.历史命令
	history [选项] [历史命令保存文件]
	-c:清空历史命令
	-w:把缓存中的历史命令写入历史命令保存文件~/.bash_history
5.输出重定向
	命令 > 文件  覆盖
	命令 >> 文件  追加
	错误命令 2> 文件  覆盖 
	错误命令 2>> 文件  追加 
	
	命令 &> 文件	正确的和错误的都写入文件(覆盖)
	命令 &>> 文件	正确的和错误的都写入文件(追加)
	命令 >> 文件1 2>> 文件2	正确的写入文件1 错误的写入文件2
	
	wc [选项] [文件名]
	-c统计字节数
	-w统计单词数
	-l统计行数
6.管道符
	命令1 | 命令2
	命令1的正确输出作为命令2的操作对象
